# Python-BDD-behave
1.关键词
-----
  * Feature （功能）
  * Scenario （场景）
  * Scenario Outline （场景大纲）
  * Given （假如）
  * When （当）
  * Then （那么）
  * Environment （环境变量）
    * 在某些特定代码执行之前运行
    * 比如在一个 Feature 开始或之后执行
    * 在一个 Scenario 开始或之后执行
    * 在一个 Step 开始或之后执行
